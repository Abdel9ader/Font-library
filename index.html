كيفية استخدام هذه الخطوط في موقعك

لإضافة أي من هذه الخطوط إلى موقعك، اتبع هذه الخطوات:

1. أضف رابط الخط في قسم head

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Tajawal:wght@300;400;700&family=Cairo:wght@400;700&display=swap" rel="stylesheet">

2. طبق الخط في CSS

body { font-family: 'Tajawal', sans-serif; direction: rtl; line-height: 1.8; } h1, h2, h3 { font-family: 'Cairo', sans-serif; }

3. مثال كامل لصفحة HTML

<!DOCTYPE html> <html lang="ar" dir="rtl"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>موقعي</title> <link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Tajawal:wght@300;400;700&family=Cairo:wght@400;700&display=swap" rel="stylesheet"> <style> body { font-family: 'Tajawal', sans-serif; direction: rtl; line-height: 1.8; } h1, h2, h3 { font-family: 'Cairo', sans-serif; } </style> </head> <body> <h1>مرحباً بكم في موقعي</h1> <p>هذا مثال على استخدام الخطوط العربية الجميلة في موقع الويب.</p> </body> </html>
